# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-04
### Added
- Initial release
- Drag & drop PDF → TIFF conversion
- Virtual printer workflow (`--latest`)
- Multipage TIFF output
- CCITT Group4 compression (MODI compatible)
- Portable ImageMagick + Ghostscript support
- Protection against partially written PDFs
